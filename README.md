# 家計簿アプリ

夫婦で使う家計簿アプリ。食費・日用品を日次で記録し、カレンダーと週次サマリーで支出を管理する。

## 技術スタック

- **Frontend**: React Native (Expo) + NativeWind
- **Backend**: NestJS (TypeScript)
- **DB**: PostgreSQL
- **Infra**: AWS (EC2 + RDS)
