# 機能一覧

## 認証

| 機能 | 説明 |
|------|------|
| ログイン | メールアドレス・パスワードでログイン |
| ログアウト | セッションを終了する |

> 1アカウントを夫婦で共有する

## 収支

| 機能 | 説明 |
|------|------|
| 収支登録 | 日付・カテゴリ・金額・メモを入力して登録 |
| 収支編集 | 登録済みの収支を編集 |
| 収支削除 | 登録済みの収支を削除 |

## カレンダー

| 機能 | 説明 |
|------|------|
| 月次カレンダー表示 | 月単位でカレンダーを表示 |
| 日別支出合計表示 | 各日の支出合計をカレンダー上に表示 |
| カテゴリ絞り込み | 食費・日用品でフィルタリング |

## 週次サマリー

| 機能 | 説明 |
|------|------|
| 週次支出合計表示 | 週単位で食費・日用品それぞれの合計を表示 |
| 予算目標設定 | カテゴリ別に予算を設定 |
| 残額表示 | 予算に対するカテゴリ別の残額を表示 |
